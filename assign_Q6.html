1. <h1> - The "h1" tag defines the most important heading on a webpage. It represents the top-level heading and is typically used for the main title or heading of a page.
  2. <p> - The "p" tag is used to define a paragraph of text. It represents a block of text content and is commonly used to structure and format paragraphs on a webpage.
  3. <a> - The "a" tag is used to create a hyperlink. It is an anchor element that can link to another webpage, a specific section within the same page, an email address, or a file to download.
  4. <img> - The "img" tag is used to embed an image into a webpage. It allows you to specify the source (URL) of the image, its dimensions, alt text (for accessibility), and more.
  5. <ul> - The "ul" tag defines an unordered list. It represents a bulleted list of items where the order of the items is not important. It is commonly used to create menus, navigation bars, or lists of related items.
