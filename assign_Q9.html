1. <strong> - Strong emphasis
    • Use to indicate strong emphasis on the text content.
    • Commonly used for highlighting important information or keywords.
    • Avoid overusing it, as it may diminish its impact if used excessively.
  2. <em> - Emphasis
    • Use to indicate emphasis on the text content.
    • Commonly used for emphasizing a word or phrase within a sentence.
    • Avoid using it solely for visual styling purposes, as it carries semantic meaning.
  3. <u> - Underline
    • Use to underline text for visual emphasis.
    • Best practice is to avoid using underlines for non-link text, as it may cause confusion with hyperlink styling.
    • Consider using other text formatting options like <strong> or <em> for emphasizing text.
  4. <del> - Deleted text
    • Use to indicate deleted or removed text.
    • Commonly used in scenarios like strikethrough for outdated or superseded information.
    • Avoid using it solely for visual styling purposes; instead, use CSS to style text with strikethrough.
  5. <sup> - Superscript
    • Use to render text as superscript, typically for mathematical notations, footnotes, or exponents.
    • Best practice is to avoid using it for general styling purposes unrelated to its semantic meaning.
    • Avoid using it for citation numbers or linking them directly to footnotes, as it may impact accessibility.
